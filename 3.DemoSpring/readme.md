# DemoSpring
This is a demo project for Spring Framework. It includes basic configurations and examples of how to use Spring in a simple application.
## Getting Started
To get started with this project, follow these steps:
1. Clone the repository:
   ```bash
   git clone
    ```2. Navigate to the project directory:
   ```bash
    cd DemoSpring
    ```
3. Build the project using Maven:
   ```bash
    mvn clean install
    ```
4. Run the application:
   ```bash
    mvn spring-boot:run
    ```
## Features
- Basic Spring configuration
- Example of dependency injection
- Simple REST controller
